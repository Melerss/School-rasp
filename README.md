# Телеграмм бот для просмотра расписания в школе
## В телеграмм боту будут слудуйщие кнопки:
* Расписание(inline) - По нажатию кнопки появятся кнопки с днями неделей
* Кнопки дней недели(inline) - По нажатию кнопки появятся уроки дня которого нажал пользователь
* Кнопки уроков - Просто для красоты
## Используем дружелюбные сообщения со смайликами
Примеры:
> Добро пожаловать в бота!😁
## Команды
* /start - выводит приветственные сообщения
* /help - список с командами
* /add - добавляет урок
## База данных
Lessens - 
* id
* weekend
* namber_lessen
* name_lessen
* time_start
* time_end
* class
## users
* telegramm_id
